# Responsive Curriculum Vitae Template

A clean, minimalist, and semantic HTML/CSS template designed for developers, designers, and professionals to showcase their academic background, technical skills, and work experience. This project serves as a structured, production-ready starting point for a digital resume or portfolio website.

## 🚀 Features

*   **Semantic Structure:** Built using clean, native HTML5 tags to ensure accessibility and high readability.
*   **Modern Typography:** Integrated with the **Inter** font family via Google Fonts for a sleek, highly legible, professional appearance.
*   **Modular Sections:** Clearly defined placeholders for Personal Info, Professional Summary, Technical Skills, Education, Work Experience, and Social/Professional Links.
*   **Highly Customizable:** Fully self-contained styles making it effortless to swap colors, tweak layouts, or add animation effects.

## 📂 Project Structure

```bash
└── project/
    └── index.html      # Main application structure and layout containing inline styling
```

## 🛠️ Built With

*   **HTML5** - Core structure and semantic markup.
*   **CSS3** - Inline styling, box-modeling, and layout definition.
*   **Google Fonts** - Typography enhancement using *Inter*.

## ⚙️ Getting Started & Installation

To run this project locally, follow these simple steps:

### Prerequisites
A modern web browser (e.g., Google Chrome, Mozilla Firefox, Brave, Safari, Microsoft Edge).

### Installation
1. Clone or download this repository:
   ```bash
   git clone https://github.com/your-username/cv-template.git
   ```
2. Navigate to the project directory:
   ```bash
   cd cv-template
   ```
3. Open `index.html` directly in your browser:
   ```bash
   # Linux/macOS
   open index.html
   
   # Windows
   start index.html
   ```

## ✒️ Customization Guide

You can easily adapt this template to suit your personal branding:
*   **Typography:** The font is loaded from Google Fonts in the `<head>` section. You can replace the Inter font URL with any font of your preference.
*   **Color Palette:** The accents currently use a vibrant green (`#00B500`) and soft blue (`#0080FF`). You can use a code editor to find and replace these hexadecimal values with your custom brand colors.
*   **Layout Adjustments:** The container features a clean frame configuration (`border: 2px solid black`). You can easily scale padding or drop the borders entirely inside the `<body>` and `.container` styling fields.
